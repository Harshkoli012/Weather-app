# Weather Application

A clean, minimalist, and responsive weather web application that allows users to fetch real-time weather details for any city globally using the OpenWeatherMap API.

## Features
* **Real-Time Data:** Fetches up-to-date temperature and weather descriptions.
* **Metric Units:** Displays temperature data natively in Celsius (°C).
* **Fully Responsive:** Centered UI layout that adapts seamlessly to desktop, tablet, and mobile screens.
* **Error Handling:** Gracefully handles empty inputs and informs users if a searched city cannot be found.

## Technologies Used
* **Frontend:** HTML5, CSS3, JavaScript (ES6+ async/await architecture)
* **API Provider:** [OpenWeatherMap API](https://openweathermap.org/)

## Live Demo Look
The interface features a centered dark-themed container holding:
1. A search input bar for entering city names.
2. An interactive "Get weather" submission button.
3. A dynamic display layout that reveals weather conditions upon successful data retrieval.
